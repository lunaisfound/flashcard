# CSE330

**Rubric approved by Cheng Li**

FlashShare is a web-based flashcard learning platform inspired by Anki, designed to make studying both personal and social. The site allows users to create, edit, and study flashcards using a spaced-repetition algorithm while also enabling them to share decks with others through unique links.
